About this project

This map tells the story of a nice full-day hike my wife and I did in the Abel Tasman National Park, New Zealand, in February of 2015. The hike is the south segment of the Abel Tasman Coast Track. To reach the trailhead in Anchorage Bay, we took a water taxi in Marahau and returned by foot, visiting Pitt Head and Te Pukatea Bay first. This is a beautiful coastal hike with wonderful vegetation, stunning views of the coast and truly amazing swimmable beaches.

Please feel free to explore the slippy map to recreate our experience and learn more about this beautiful location. Tracks, markers and photos are clickable. The controls in the top left part of the map provide for zooming, base map and layer display control, and access to an interactive elevation profile widget.

I developed the application using open source web development tools, frameworks and freely available base map tile services. It should work ok on tablets and phones, but it is best viewed from a PC using a modern browser.

This application is my final assignment deliverable ("Storytelling with maps") for the Maps and the Geospatial Revolution MOOC course by Dr. Anthony C. Robinson, that I took between March and April of 2015.

Building blocks

Core mapping framework:
  • The powerful Leaflet Javascript library
Components and plugins:
  • Bootstrap for mobile friendly navigation bar and dialogs
  • jQuery mobile-friendly Javascript utility library
  • Leaflet.Elevation to display trail elevation profile
  • Leaflet.MakiMarkers to display nice Maki icons in markers
  • Lightbox2 to display pictures and slideshow
  • Omnivore plug-in to consume GPS tracks in .GPX format
Base maps:
  • Thunderforest outdoors map that shows topographic features and information relevant to hiking
  • ESRI World Imagery to show satellite and aerial imagery
GPS Tracks:
  • Recorded on a Garmin Oregon 450 GPS unit
