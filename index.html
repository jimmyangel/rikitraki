About this site

Welcome to my personal hiking log. Feel free to use the slippy map to explore our hiking experiences and learn more about their locations. Tracks, markers and photos are clickable. The controls at the top left part of the map provide for zooming, base map and layer display control. The control at the bottom left is an interactive elevation profile widget. The track label at the top right provides track information and a slide show (if pictures are available.)

This is a work in progress. Please forgive the bugs and omissions. I will be adding tracks to the site and continue testing and enhancing it over time.

Building blocks

This project uses free and open source components and data as shown below.

Core mapping framework:
  • The powerful Leaflet Javascript library
Components and plugins:
  • Bootstrap for mobile friendly navigation bar and dialogs
  • jQuery mobile-friendly Javascript utility library
  • Leaflet.Elevation to display trail elevation profile
  • Leaflet.MakiMarkers to display nice Maki icons in markers
  • Lightbox2 to display pictures and slideshow
  • Omnivore plug-in to consume GPS tracks in .GPX format
  • Leaflet Spin plug in to display wait spinner
  • publicdomainvectors.org public domain icons
Base maps:
  • Thunderforest outdoors map that shows topographic features and information relevant to hiking
  • Open Street Map that shows roads, labels and geographic features to show satellite and aerial imagery
  • ESRI World Imagery to show satellite and aerial imagery
  • US Geological Service official US topographic map that shows satellite and aerial imagery (US only)
